== Installation ==
This section describes how to install the plugin and get it working.
1. Upload `woonumdays.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If user is Editor or above, they will see a Selection button in the Editor
4. YOU MUST INSTALL THE COMMENTS TEMPLATE IN `templates` folder - or alter your current comments.php file.

== Frequently Asked Questions ==
Q.: My normal blog posts do not work correctly and I can not see my comment sin them.
A.: In order to have a blog with comments you will have to install a second instance of WordPress. This plugin creates two new custom comment types and they are not compatable with the standard WP comments.

Q.: Can I change the background colors of individual boxes?
A.: There is a chart in your admin page with the names of all the selectors. If you know CSS then you can use any CSS editor to add
your background colors to.
